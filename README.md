# Zenith - Task Tracker 
Live Link - https://mohammedismailcn.github.io/task-tracker

A simple and easy-to-use task tracker created for my own purpose and daily productivity needs.

This project is a high-fidelity single-page web application designed to help manage tasks efficiently with real-time deadline alerts, a built-in notepad, and a drawing board for quick sketches and ideas.

## Why I Built This
I created this project to have an all-in-one productivity workspace where I can:

- Track tasks with exact deadline countdowns
- Get sound alerts for important deadlines
- Write and save quick notes instantly
- Sketch ideas, diagrams, or plans on a built-in drawing board
- Keep everything in one place without needing multiple apps

## Features

### Task Management
- Create, edit, delete, and complete tasks
- Set exact date & time deadlines
- Live countdown timers
- Priority levels and categories
- Search and sort tasks
- Sound alarm notifications using Web Audio API

### Notepad
- Persistent notes with auto-save
- Multiple note support
- Quick note management
- Embed drawings directly into notes

### Draw Board
- Pen / Brush tool
- Line, Rectangle, Circle tools
- Eraser
- Custom colors & brush sizes
- Undo / Redo
- Clear canvas
- Export drawings as PNG

### Dashboard
- Task statistics overview
- Pending/completed task counts
- Upcoming deadline insights
- Progress visualization

### Settings
- Theme controls
- Audio settings
- Alarm testing

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Web Audio API
- LocalStorage
- HTML Canvas API

## Storage
All application data is stored locally in the browser using LocalStorage. No backend or server setup is required.

- User authentication
- Export/import task data
